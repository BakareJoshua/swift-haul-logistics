# Swift Haul Logistics Performance Dashboard

## Project Overview

This project analyzes logistics performance for Swift Haul Logistics using Power BI. The dashboard helps stakeholders understand shipping value, shipping cost, shipment volume, vendor performance, regional performance, transport mode cost, warehouse activity, and customer ratings.

## Tools Used

- Power BI
- Microsoft Excel
- DAX

## Key Metrics

- Total Shipping Value: 23M
- Total Shipping Cost: 298M
- Shipment Count: 1.5K
- Average Profit Margin: 0.19

## Key Insights

- Lagos recorded the highest shipping value and shipping cost.
- Zone D had the highest shipment count among warehouse zones.
- Vendor shipment counts were fairly close, showing balanced vendor activity.
- Air and road transport contributed the highest shipping costs.
- Customer ratings were highest at rating level 5, showing strong customer satisfaction.
- Agro-related products had the highest average profit margin.

## Recommendations

- Review shipping costs in Lagos because it has the highest cost.
- Monitor air and road transport expenses to improve cost control.
- Maintain strong service quality to keep customer ratings high.
- Study Zone D operations to understand why it handles the highest shipment volume.
- Focus on product categories with stronger profit margins.
